Bandyopadhyay-Mehler Hydropathy Plot

To generate hydropathy plot, please follow the on-screen instructions.
  • Type in the amino acid sequence of the protein in the box labelled "Amino acid sequence"
  • For eg: If you want to generate plot for the protein with PDB ID: 2CZS, type
    MVSGEVRTKKVPLDTNHKRFYDAFAQGAGKLDLDRQCVECHHEKPGGIPFPKNHPVKPADGPMRCLFCHKFKLEHHHHHH
    only without any punctuation marks.
  • Type in window size without any punctuation marks in the box labelled window size
  • Once you finish the above steps, click on "Get plot" button.
